What is PHP?

PHP is a powerful server-side scripting tool which can be embedded into HTML and allows us to do many things. What distinguishes PHP from something like client-side JavaScript is that the code is executed on the server, generating HTML which is then sent to the client. The client would receive the results of running that script, but would not know what the underlying code was. You can even configure your web server to process all your HTML files with PHP, and then there's really no way that users can tell what you have up your sleeve.

What can you do?

PHP can be used to load in additional HTML files and elements. It is a general purpose scripting language, although it's main purpose has been for web development. The three main areas where PHP are used are in server-side scripting, command line scripting, and for writing desktop applications. In server-side scripting, PHP can be used for collecting sform data, generating dynamic page content, or sending and receiving cookies.
