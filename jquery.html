What is jQuery?

While the name jQuery might seem daunting, jQuery itself is simply a javascript library. It works to simplify and streamline common functions. In fact, its motto closely reflects this thinking: Write less, do more. It allows for rapid web development by simplifying HTML document traversing, event handling, animating, and Ajax interactions. Most notably, it makes it easier to select and traverse dom elements, utilize event handlers, and perform Ajax calls. Ajax, or asynchronus javascript and XML, is "the use of the XMLHttpRequest object to communicate with server-side scripts". However, it is especially important due to its asyncrhonus nature, which allows us to update certain portions of a page without needing to refresh the entire page.

What can you do?

jQuery is highly capable, like javascript. It can be used to select elements in the DOM, as well as respond to user interaction and actions. By utilizing Ajax calls, jQuery can be used to updae specific page sections based on user interaction without needing to reload the entire page.
