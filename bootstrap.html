What is Bootstrap?

Bootstrap is a HTML, CSS, and JS framework which can be used to develop webpages. It provides many common elements and forms prebuilt and ready to use. In addition, it is designed to be mobile-friendly, meaning that it is capable of being used on devices with smaller screen sizes, such as phones and tablets. The responsive design of bootstrap is becoming increasingly important in our increasingly mobile society. In addition to HTML and CSS, bootstrap comes with several JavaScript components in the form of jQuery plugins. They provide additional user interface elements such as dialog boxes, tooltips, and carousels.

What can you do?

Bootstrap is packed to the brim with extremely useful features and abilities. It includes a 12-column grid with which to structure pages around, Javascript and jQuery plugins fo user interaction, and typography and forms.
